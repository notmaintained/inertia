inertia by sandeepshetty

Inertia is a Resource-Oriented Programming (ROP) platform for building web applications and frameworks in PHP that provides a module level abstraction based on REST.

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/sandeepshetty/inertia

get the source code on GitHub : sandeepshetty/inertia
