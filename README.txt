
What is Inertia?

Inertia is a Resource-Oriented Programming (ROP)[1] platform for building web
applications and frameworks in PHP[2] that provides a module level abstraction
based on REST[3].

Inertia is free software[4] and distributed under the terms of the GNU General
Public License[5].

Check out http://inertia.sourceforge.net/ for more information.
